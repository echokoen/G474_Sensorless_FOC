# STM32G474 Sensorless PMSM FOC

基于 STM32G474 的无感 PMSM FOC 控制项目，从零实现完整控制链路。

---

## 📌 功能

- 无感启动（I/F 开环）
- 磁链观测器 + PLL
- OPENLOOP → RUN 切换（switchover）
- dq 电流环
- 速度环闭环控制
- 三电阻电流采样
- SVPWM 驱动

---

## ⚙️ 硬件

- MCU：STM32G474
- 母线电压：24V
- 电流采样：三下桥电阻
- PWM：TIM1（中心对齐）
- ADC：注入通道同步采样

---

## 🔌 引脚说明（核心）

### 🔹 PWM 输出（TIM1）

| 相位 | 引脚 | 定时器   |
| ---- | ---- | -------- |
| U相  | PA8  | TIM1_CH1 |
| V相  | PB13 | TIM1_CH2 |
| W相  | PB14 | TIM1_CH3 |

（互补输出用于驱动下桥）

---

### 🔹 电流采样（ADC）

| 相位             | ADC通道   | 引脚 |
| ---------------- | --------- | ---- |
| U相电流          | ADC2_IN3  | PA2  |
| V相电流          | ADC2_IN4  | PA3  |
| W相电流ADC12_IN9 | ADC12_IN9 | PC3  |

说明：
- 采用三下桥电阻采样
- 实际采样两相，第三相通过 `Ia + Ib + Ic = 0` 计算得到

---

### 🔹 母线电压采样

| 信号 | ADC通道  | 引脚 |
| ---- | -------- | ---- |
| Vbus | ADC1_IN1 | PA0  |

---

### 🔹 ADC 触发

- 由 TIM1 触发 ADC 注入转换
- 与 PWM 同步采样
- 采样点位于 PWM 中心区域

---

## 📊 当前状态

- 已实现无感启动
- 可稳定进入闭环运行
- 最高转速：2200~2500 rpm

---

## ⚠️ 已知问题

- 高速性能未达到额定转速（3000 rpm）
